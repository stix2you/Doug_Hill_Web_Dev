ChatApp Project Case Study

The ChatApp is a mobile app which provides users with a very basic CHAT interface, and options to share images and their location. It features Asynchronous offline storate on the client side, and a real-time Firebase/Firestore database on the server side. The app is built using React Native, Expo, and Google Firestore.

  • Technologies Used:
    HTML5 CSS JavaScript (ES6+) jQuery AJAX Bootstrap PokeAPI IIFE (Immediately Invoked Function Expression)
  • Initial Setup
    HTML and CSS
    The initial setup focused on creating a basic HTML structure that incorporates the necessary Bootstrap styles and custom CSS. I used a fixed-top navigation bar with the Pokédex logo and a responsive Pokémon list section. The UI leverages CSS and Bootstrap for responsive design, custom styles and theme consistency.
  • Implementing JavaScript Logic
    Module Pattern with IIFE
    I encapsulated all JavaScript logic in an IIFE to avoid polluting the global scope. The pokemonRepository object serves as the primary interface for interacting with Pokémon data.
